# Hollow Knight Silksong Save Analyser

This is a tool which checks your save file's progress!

Made in SvelteKit by [@BlueOrcaz](https://github.com/BlueOrcaz)

Website: https://silksong-save-viewer.vercel.app/

## Tracks:
- Save Information
  - Save Version
  - Game Completion Percentage
  - Time Played
  - Mask Shards
  - Silk Hearts
  - Rosaries
  - Shell Shards
- Defeated Bosses
- Unlocked Crests
- Needle Upgrades
  - Crafting Kits
  - Tool Pouches
  - Mask Shards
  - Spool Fragments
  - Abilities
  - Silk Skills
- Completed Wishes/Quests
- Unlocked Maps
  - Bellway Stations
  - Ventrica Stations
- Unlocked Tools
- Collectables
  - Fleas
  - Simple Keys
  - Misc. Keys
  - Craftmetal
  - Pale Oil
  - Memory Lockets
  - Mossberries
  - Bone Scrolls
  - Rune Harp
  - Weaver Effigy
  - Mementos
  - Choral Commandments
  - Psalm Cylinders
  - Arcane Egg
- Gamepass/macOS/Linux File Upload Support
- Hunter's Journal
  - Kill Count
  - Checklist
  - Bugs Encountered
